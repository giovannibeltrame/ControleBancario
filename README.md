# ControleBancario
Java Web Application for simulation of a banking system.

### Technologies used:
- Java Development Kit 7
  - Servlet
  - JSP
  - JSTL
  - JDBC
- CSS
  - SB Admin Template
  - Bootstrap
  - Font Awesome
- JavaScript
  - jQuery
  - morris.js
  - Raphael.js
- Oracle Database Express Edition
- Jasper Reports
